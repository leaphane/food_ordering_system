# food_ordering_system

## 1. What is Spring Boot?
> *It's a java framework that is used to spring applications.*
## 2. What is Maven?
> *It's a dependency management tool and build automation tool and provides a standard project structure.*
## 3. What is the purpose of pom.xml?
> *Maven's central configuration file used to manage dependencies, plugins, build configuration, and project metadata..*
## 4. What is the purpose of application.properties?
> *To externalize configurations and set runtime values .*
## 5. What does @SpringBootApplication do?
> *It automatically configures the application and sets up everything needed for it to run when you start it.*
## 6. Why do developers use dependency management tools such as Maven?
> *To automate the build lifecycle, manage dependencies, and enforce a consistent project structure across projects.*
## 7. What is a REST API?
> *web-based interface that allows systems to communicate and perform CRUD operations using HTTP methods.*
## 8. What is JSON?
> *A data format used to send and receive structured data*
## 9. What is Dependency Injection
> *a design pattern where objects are provided their dependencies instead of creating them themselves, making code loosely coupled and easier to maintain and test.*
